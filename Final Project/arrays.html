Arrays

Arrays are data structures used to store ordered collections.

Declaring an array

There are two common ways for creating an empty array:

let arr = new Array();
let arr = [];

Almost all the time, the second syntax is used. We can supply initial elements in the brackets:

let fruits = ["Apple", "Peach", "Pumpkin"];

Array elements are numbered, starting with zero.
We can get an element by its number in square brackets:

let fruits = ["Apple", "Peach", "Pumpkin"];

alert( fruits[0] ); // Apple
alert( fruits[1] ); // Peach
alert( fruits[2] ); // Pumpkin

Reeplacing an element:

fruits[2] = 'Banana'; // now ["Apple", "Peach", "Banana"]

Adding a new element to the array:

fruits[3] = 'Lime'; // now ["Apple", "Peach", "Banana", "Lime"]

The total count of the elements in the array is its length:

let fruits = ["Apple", "Peach", "Banana", "Lime"];
alert( fruits.length ); // 4

Alert can also be used to show the whole array.

let fruits = ["Apple", "Peach", "Banana", "Lime"];
alert( fruits ); // Apple,Peach,Banana, Lime

An array can store elements of any type. For example:

// mix of values
let arr = [ 'Apple', { name: 'John' }, true, function() { alert('hello'); } ];

// get the object at index 1 and then show its name
alert( arr[1].name ); // John

// get the function at index 3 and run it
arr[3](); // hello

Iterating through an array

A for loop can be used to iterate the idices:

let arr = ["Apple", "Orange", "Pear"];
for (let i = 0; i < arr.length; i++) {
  alert( arr[i] );
}
